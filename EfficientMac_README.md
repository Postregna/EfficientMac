# EfficientMac

A collection of tools I'll continue to add to.

## Tools

### autolock
Hold Right Shift + Right Ctrl for 3 seconds to lock your screen. Quits all open apps, empties the trash, and plays a sound so you know when to let go.

Runs automatically at login.

### AppMover
Watches for a target app to open and automatically moves it to your second monitor. Defaults to VS Code on the left monitor but works with any app.

Runs automatically at login.
